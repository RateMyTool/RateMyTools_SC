# RateMyTools

A community-driven review platform for UH Manoa students to rate and review educational tools and AI resources.

For complete documentation, visit: https://ratemytool.github.io/RateMyToolPage/

## Quick Start
```bash
npm install --legacy-peer-deps
cp .env.local .env
npx prisma generate
npm run dev
```

Visit http://localhost:3000
